# NotifySync - Notification Scheduler App

NotifySync is a modern, attractive Android application built with **Jetpack Compose** and **Material 3**. It allows users to create, schedule, and manage notifications with a focus on a beautiful user experience.

## ✨ Features

- **Secure Onboarding**: Attractive Login and Sign-Up screens with email verification.
- **Dynamic Dashboard**: A central hub to navigate through all app features.
- **Notification Creation**: Sleek form to schedule notifications with custom titles, messages, and priority levels.
- **Schedule Manager**: View and manage all upcoming scheduled tasks in one place.
- **Notification History**: Keep track of all sent, pending, and failed notifications.
- **Priority Settings**: Customize how different priority levels behave.
- **App Settings**: Personalize your experience with Dark Mode and more.

## 🛠 Tech Stack

- **UI**: Jetpack Compose
- **Design**: Material 3
- **Navigation**: Compose Navigation
- **Architecture**: MVVM (ViewModel & Shared State)
- **Language**: Kotlin

## 🚀 Getting Started

1. Clone the repository.
2. Open in Android Studio.
3. Sync Gradle and run the app on an emulator or physical device.
